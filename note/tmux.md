## tmux

### session

```bash
创  建          tmux new -s <session-name>
进  入				 	tmux attach -t  <session-name>
命  名 			    ctrl + x $
切  换		      ctrl + x ( or )
离  开          ctrl + x d
列  表          ctrl + x s
关  闭				  ctrl + d
```


### window
```bash
创  建          ctrl + x c
列  表          ctrl + x w
切  换          ctrl + x n or p
命  名          ctrl + x ,
关  闭          ctrl + x &
```


### panel
```bash
左右切割       ctrl + x %
上下切割       ctrl + x “
移   动       ctrl + x 方向键
关   闭       ctrl + x x
放大/还原      ctrl + x z
显示时间       ctrl + x t
调整大小       ctrl + x alt+方向键
```
