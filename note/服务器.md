### sz 和 rz 的使用
**上传**

```bash
# 输入 rz
rz
# 键盘输入 Ctrl+2, 进入 zssh>
# cd 切换到上传路劲
cd ~/data
# sz 指定上传文件
sz cli_test

##################
# 输出过程
##################

root@localhost:~# rz
 # waiting to receive.**▒B0100000023be50
zssh > pwd
/home/jingle/data/jhgame/server_git
zssh > cd ~/data
zssh > ls
cli_test  java.learn  jhgame  my_query.sql  Pictures  py.learn  share  sygame  typora  wechat_back  wechat_stat  work
zssh > sz cli_test 
Sending: cli_test
Bytes Sent:      9   BPS:18                              

Transfer complete
root@localhost:~# ls
cli_test  install.sh  snap  srv_test  tcp.sh  v2ray_info.inf  virt-sysprep-firstboot.log
```

**下载**

```bash
# sz file
sz srv_test
# 键盘输入 ctrl+2, 进入 zssh>
# cd 切换到保存路径
cd ~/data
# rz 接受文件
rz

##################
#输出过程
##################

root@localhost:~# ls
install.sh  snap  srv_test  tcp.sh  v2ray_info.inf  virt-sysprep-firstboot.log
root@localhost:~# sz srv_test 
 #▒B00000000000000
zssh > cd ~/data
zssh > pwd
/home/jingle/data
zssh > rz

Receiving: srv_test                                                  
Bytes received:       9/      9   BPS:56                    

Transfer complete
```
